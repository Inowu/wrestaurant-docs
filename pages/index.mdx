API REST para integrar pedidos, menús y ventas en tiempo real con Soft Restaurant.

Tu sucursal sigue operando localmente; nosotros exponemos solamente los datos que tú autorizas, en tiempo real, con autenticación por **API key**.

## ¿Qué puedes hacer?

- Consultar **productos** y **estaciones** de cada sucursal sin tocar la base de datos local.
- Obtener resultados en formato JSON con paginación predecible.
- Autenticar cada llamada con un **API key** emitido desde tu panel de administración.
